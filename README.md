# py-funny-parkour
sussy parkour game made 4 years ago

Just execute `main.py`

## Modules required:

- `pygame 2.1.2+`. that's all

## Features:

-  29 built-in levels

-  4 ~~sussy~~ game modes (`1~10 lvl`, `11~20 lvl`, `21-25 lvl`, `26-29 lvl`)

## Control:

-  `[←]` move left

-  `[→]` move right
  
-  `[↑]` jump
  
-  `[↓]` skip level
  
-  `[space]` decelerate
