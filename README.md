# py-funny-parkour
sussy parkour game made 4 years ago


Features:

-  29 built-in levels ~~that's all~~

------------

Control:

-  `[←]` move left

-  `[→]` move right
  
-  `[↑]` jump
  
-  `[↓]` skip level
  
-  `[space]` decelerate
